# MineWeb-BOT
MineWeb BOT is a bot for MineWeb that can do much stuff like keeping your server online 24/7.

## Features

1. Keep your server alive 24/7
2. Use commnands
3. Manage the bot with a cool web interface

## How to host the bot (using Render)

1. Create a new Web Service
2. after the web service is live, link the ip address of the server to the bot
3. Done!
